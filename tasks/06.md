## Task

Create a chatbot that calls the `/ai` API route (do not use Orval).

## Technical Requirements

- Use Vercel AI SDK.

``tsx

const { messages, sendMessage, status } = useChat({
transport: new DefaultChatTransport({
api: "/ai",
}),

});

``
- Store the chat opening status in the URL and also store an initial message that will be sent to the chat automatically. Use the "nuqs" library to do this. Example: localhost:3000?chat_open=true&chat_initial_message=message
- Open the chatbot by clicking the central button in the navigation bar (with the Sparkles icon).

- Open the chatbot by clicking the "?" button in @exercise-card.tsx and the initial message (passed via URL) should be "How to perform exercise X correctly?"

- Use the "streamdown" library to format the chatbot messages.

- When opening the chatbot and there are no messages, display the following suggested messages: "Build my workout plan".

- Be 100% faithful to Figma. See https://www.figma.com/design/icjlmwVThvfpiYYXFld8gG/FIT.AI?node-id=131-493&m=dev
